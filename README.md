# ArduinoPlayPianoTiles
It's a arduino based robot able to play piano tiles game.
Used Hardwares:
  4 relay switches
  4 22k resisters
  4 LDR sensord
  1 Arduino UNO board
  1 breadboard
  4 aluminium coins
  
  
Here first you need to check sensivity of your LDR sensor.
for that use test_ldr code.
